# 📰 React Firebase News App

A modern single-page news application built with **React**, **Vite**, **Firebase Authentication**, **TailwindCSS**, and **DaisyUI**. The app supports protected routes, dynamic news loading, lazy loading, and user authentication.

---

## 🚀 Features

- 🔐 **Firebase Auth** (Email/Password login)
- 📄 News listing and detailed single news page
- 🔒 Protected/private routes with context
- 🎨 Styled with TailwindCSS + DaisyUI
- ⚡ Powered by Vite for fast development and hot reload
- 🧠 Authentication handled using React Context API
